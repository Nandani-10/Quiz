⚡ One-time Firebase Setup

To work across all devices, this quiz needs a free Firebase Realtime Database. Takes about 2 minutes:

1
Create free project
Go to console.firebase.google.com → New Project (no billing needed)
2
Enable Realtime Database
Build → Realtime Database → Create database → Start in test mode
3
Get your config
Project Settings → Your apps → Add Web App → copy the firebaseConfig object
4
Paste into this file
Open quiz-app.html in a text editor → find const firebaseConfig → replace the values
